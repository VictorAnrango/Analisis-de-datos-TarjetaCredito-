Modelo de maching learning:

Ayuda con la calidad del Vino teniendo en cuenta todos sus parametros 
Y ayuda a predecir el fraude y no fraude de una transaccion de un pago con una tarjeta de credito.

El VinoRojo.py esta completo, en el dataset encuentra tanto para Vino Blanco como para Vino Rojo que es el utilizado en este caso.
El proyecto FraudeCredi2.py, esta completo y funciona al 100%, el proyecto FraudeCrediticio.py esta todavia en el modelado.

Dataset VinoRojo:
https://archive.ics.uci.edu/dataset/186/wine+quality

Dataset Fraude Crediticio:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data
